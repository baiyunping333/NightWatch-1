# Magic

Discussion related to in-game magic system.


- Elements:
  - Earth
  - Wind
  - Fire
  - Water
  - Light
  - Dark
  - Acid
  - Lightning
  - Arcane
  - Life
  - Death
  - Primal
  - Chaos
  
- Base spells are single element. Offers greater damage (for greater cost?) by mixing.
- Mix elements
  - *Ex: Mix 2 fire + 1 wind = fireball*
  - *Ex: Mix 2 fire + 1 earth = scorched earth*
